Mafia Streets

Underworld Syndicate RPG - Rebranded and enhanced from Mumbai Mafia

Implemented Features

1. Rebranding & UI Aesthetics

  • Animated Logo: Replaced the static logo with a letter-by-letter CSS-animated title. Individual letters float, jump in sequence, and spin on cursor hover. It features a neon shine and radial underworld pulse glow.
  • Movable Character Selection: Interactive selection cards for choosing between Maya and Vikra:
    • 3D Tilt Hover: Hovering over cards tilts them dynamically in 3D perspective space based on the mouse pointer position.
    • Breathing Float: Cards float passively with sinusoidal waves (each character has different wave delays for natural movement).
    • Rubber-Band Dragging: Cards can be clicked/touched and dragged around the screen freely, springing back to their slots with a physical snap transition on release.

2. Gameplay Mechanics

  • Maya (The Mastermind): Starts with Lathi (equipped), 60 max energy, base attack 12, defense 15. Special: Calculated Strike (bypasses defense, double damage, restores +10 energy, 3-turn cooldown).
  • Vikra (The Enforcer): Starts with Lathi (equipped), 120 max HP, base attack 18, defense 8. Special: Underworld Rage (+50% attack power for 2 turns, +15 HP instant heal, 3-turn cooldown).
  • HTML5 Synthesized Audio: Built a synthesized sound system using the browser's native Web Audio API for real-time retro game sound effects (Hit, Heal, Level Up, Victory, Defeat, and Bribe).
  • Corrupt Cop Street Encounter: Territory raids have a 25% chance of triggering Inspector Shinde, demanding a ₹80 bribe. Pay to escape safely (gain +30 EXP) or fight him in a mini-boss battle.
  • Gossip Ticker: Scrolling underworld news marquee footer displaying live syndicate gossip.

How to Run & Build

Local Browser Testing

To run the developer server locally:

# Prepend node-portable to path $env:PATH = "C:\Users\ANAMIKA\Desktop\Game_APP\node-portable;" + $env:PATH # Navigate to project and run browser platform cd C:\Users\ANAMIKA\Desktop\Game_APP\mumbai-mafia node C:\Users\ANAMIKA\Desktop\Game_APP\node_modules\cordova\bin\cordova run browser

Rebuilding the Android APK

To compile the project from source:

# Include Gradle, OpenJDK 21, and Android SDK tools in PATH $env:PATH = "C:\Users\ANAMIKA\Desktop\Game_APP\node-portable;C:\Users\ANAMIKA\Desktop\Game_APP\jdk21\jdk-21.0.11+10\bin;C:\Users\ANAMIKA\Desktop\Game_APP\gradle\gradle-8.5\bin;C:\Users\ANAMIKA\AppData\Local\Android\Sdk\cmdline-tools\latest\bin;C:\Users\ANAMIKA\AppData\Local\Android\Sdk\platform-tools;" + $env:PATH # Set Home paths $env:JAVA_HOME = "C:\Users\ANAMIKA\Desktop\Game_APP\jdk21\jdk-21.0.11+10" $env:ANDROID_HOME = "C:\Users\ANAMIKA\AppData\Local\Android\Sdk" # Compile Android build cd C:\Users\ANAMIKA\Desktop\Game_APP\mumbai-mafia node C:\Users\ANAMIKA\Desktop\Game_APP\node_modules\cordova\bin\cordova build android

Challenges & Resolutions

  • Missing JDK Compiler (jlink): Resolved by downloading and configuring a portable OpenJDK 21 distribution, resolving compilation errors caused by the headless RedHat JRE.
  • Missing Gradle Build Daemon: Configured a portable Gradle 8.5 installation in the PATH variables.
  • Android API Targets: Installed target API Platform 36 and Command-line Tools via the `android` CLI tool.
